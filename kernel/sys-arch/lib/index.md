# Libraries

This section describes the libraries used in the Kyronix ecosystem.

## Kernel Libraries

- [Frigg](frigg.md) -- C++ utility library (template helpers, smart pointers)
- [Bragi](bragi.md) -- IPC serialization protocol
- [Hel](hel.md) -- Helio system call interface
- [Helix](helix.md) -- Helix async I/O primitives
- [libasync](libasync.md) -- Asynchronous execution framework

## User-Space Libraries

- [mlibc](libc/mlibc.md) -- musl-based C standard library
- [Syscall Wrappers](libc/syscalls.md) -- libc syscall wrappers
