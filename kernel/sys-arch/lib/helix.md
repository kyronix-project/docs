# Helix

This document describes the Helix async I/O library.

## Overview

Helix provides asynchronous I/O primitives for the Managarm-derived components. It enables non-blocking operations on kernel objects.

## Features

- Asynchronous handle operations
- Event loops
- Promise/future patterns
- Queue-based operation submission

> **Note:** Helix is a dependency inherited from the Managarm project.
