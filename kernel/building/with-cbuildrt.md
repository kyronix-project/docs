# Building with cbuildrt

This document describes how to build the Kyronix kernel using cbuildrt. It is the child of [Building](index.md).

## Overview

cbuildrt is a container-based build runtime that provides reproducible build environments. The Kyronix kernel build system supports cbuildrt as an alternative to Docker or Podman for containerized builds.

## Status

This document is a placeholder. The cbuildrt integration is not yet present in the kernel source tree. See [with Docker/Podman](with-docker.md) for the current containerized build method, or [without containers](without-containers.md) for native builds.
